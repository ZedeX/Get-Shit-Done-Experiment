# 实验计划: T1

## 任务信息
- **任务编号**: T1
- **场景类型**: code_development (代码开发)
- **复杂度**: simple (简单)
- **任务名称**: PNG到WebP批量转换脚本
- **任务描述**: 编写一个Python脚本，实现将指定目录下所有PNG图片批量转换为WebP格式，保留原始文件名

## 输入资源
- 无特殊输入资源 (需要时创建测试PNG图片)

## 输出要求
- **格式**: Python脚本
- **功能特性**:
  1. 递归遍历目录
  2. 支持指定输出目录
  3. 保留原始文件名结构
  4. 可配置WebP质量参数

## 验收标准
1. 脚本可正常运行
2. PNG文件正确转换为WebP
3. 文件名保持一致
4. 转换质量可配置
