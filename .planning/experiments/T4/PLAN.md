# 实验计划: T4

## 任务信息
- **任务编号**: T4
- **场景类型**: data_analysis (数据分析)
- **复杂度**: simple (简单)
- **任务名称**: 销售数据分析
- **任务描述**: 分析提供的销售数据CSV文件，计算各产品类别的总销售额、平均单价、销量排名

## 输入资源
- `input_data/sales_data_T4.csv` - 销售数据CSV文件

## 输出要求
- **格式**: 分析报告 + Python脚本
- **指标**:
  1. 各类别总销售额
  2. 各类别平均单价
  3. 销量排名
  4. 月度趋势

## 验收标准
1. 数据计算准确
2. 报告清晰易读
3. 可视化图表直观

## 执行步骤概览

### GSD模式步骤
1. 数据探索和理解
2. 分析计划制定
3. 数据清洗和预处理
4. 编写分析脚本
5. 计算各项指标
6. 生成可视化图表
7. 撰写分析报告
8. 验证分析结果
9. 文档整理

### 非GSD模式步骤
1. 读取CSV数据
2. 进行基本计算
3. 生成简单报告
